# kakoune-web

npm i && npm start

## ROADMAP
- UI
  - receives stdin from kak
  - sends back keystrokes

  - shadow client
    - displays UI overlays (dropdown etc)  seamlessly with the editing client
    - pipes to kakoune
    - autocompletion
    - JS evaluation
      - example: // x = 13; 
  - Elm-History based macros (like slo-mo videos)
    - https://github.com/elm-community/undo-redo

## Notes
